# Smart Fridge Application

Java application to manage a fridge: inventory, recipes and shopping list.
Recipe suggestions use the Spoonacular API.

## Stack

- Java
- SQLite (storage)
- Spoonacular API (recipes)

## Features

- Manage fridge inventory.
- Recipe suggestions based on available ingredients.
- Shopping list.

## Build and run

Open the project in your Java IDE, then build and run the main class. A
Spoonacular API key is required for the recipe features.
